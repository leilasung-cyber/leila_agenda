# Leila Portal

PC와 휴대폰에서 사용할 수 있는 설치형 웹앱(PWA)입니다. 현재 위치의 오늘 날씨와 외출 준비 제안도 함께 보여줍니다. 일정, 기한 있는 할 일, 언젠가 할 일, 쇼핑리스트, 아이·가족 항목을 자연어로 입력하면 제목·위치·날짜·시작/종료 시간으로 나누어 보여주고, 등록 전에 각 항목을 수정할 수 있습니다.

## 실행

PowerShell에서 이 폴더로 이동한 뒤 실행합니다.

```powershell
.\serve.ps1
```

브라우저에서 `http://localhost:4173`을 엽니다. 데이터는 현재 브라우저에 저장됩니다.

## Google Calendar 연결

1. Google Cloud Console에서 프로젝트를 만들고 Google Calendar API를 활성화합니다.
2. OAuth 동의 화면을 구성합니다.
3. 웹 애플리케이션용 OAuth 클라이언트 ID를 만들고 앱 주소를 승인된 JavaScript 원본에 추가합니다.
4. 앱의 설정 화면에 클라이언트 ID를 입력하고 Google 계정을 연결합니다.

연결하지 않아도 자체 캘린더로 사용할 수 있으며, `Google에 추가`를 누르면 미리 채워진 Google Calendar 등록 화면이 열립니다.

## 기기 간 동기화

현재 MVP는 로컬 우선 버전입니다. 일정은 같은 Google 계정으로 볼 수 있지만, 할 일·언젠가·쇼핑의 자동 동기화에는 온라인 저장소와 로그인을 추가해야 합니다.
